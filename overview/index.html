mvtest is based on pytest and includes the following standard modules to help perform a full-fledged automated QA on MontaVista Linux distribution.

paramiko - for running commands on remote host.

pexpect - for interactive testing.

pytest-html - for generating HTML reports.

colorlog - for log formatting with colors in console.

ansi2html - convert text with ANSI color codes to HTML.

sphinx - for automatic documentation generation.
