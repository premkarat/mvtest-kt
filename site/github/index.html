Setting up mvtest repo in github

  • Setup mvtest git and authenticate with github from git
  • Create mvtest repo in github without initializing the repository with a README
  • git remote add origin https://github.com/MontaVista-OpenSourceTechnology/mvtest.git
  • git push -u origin master

Contributing code to mvtest
